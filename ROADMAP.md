# Roadmap

Planned features and improvements for **YouTube Music for Kodi**. Items are ordered roughly by priority, not by timeline.

## Short term

- [ ] Repository add-on (`repository.ypoulis`) for one-click install + auto-updates
- [ ] Better playlist management (reorder / delete / edit)
- [ ] Improved local cache for artwork and metadata

## Medium term

- [ ] Resume playback across sessions
- [ ] More language translations for the add-on UI
- [ ] Better handling of mixed personal / brand accounts

## Longer term

- [ ] Smarter quality selection (auto / network-aware)
- [ ] Optional integration with the Kodi music library (artwork scrape)
- [ ] Skin-friendly metadata for richer Now-Playing screens

Have an idea? Open a [feature request](https://github.com/ypoulis-hub/kodi-youtube-music/issues/new?template=feature_request.md) or start a thread in [Discussions](https://github.com/ypoulis-hub/kodi-youtube-music/discussions).
